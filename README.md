# TimeReporter
time reporter	时间播放器

主要为了提醒自己时间

## Basic

* 基于nw.js vue.js
* 基于本地时间
* 基于百度TTS http://tts.baidu.com/text2audio?lan=zh&ie=UTF-8&spd=2&text=
